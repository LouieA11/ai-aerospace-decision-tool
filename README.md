# ai-aerospace-decision-tool
AI-based decision support tool for evaluating aerospace applications

# AI Aerospace Decision Support Tool
This project evaluates different applications of artificial intelligence in aerospace, such as satellite image analysis, aircraft fault detection, and autonomous navigation.

# Purpose
The goal is to help identify which AI applications provide the best balance of impact, risk, complexity, and feasibility.

# Features
- Ranks aerospace AI applications using a scoring model
- Allows comparison of impact vs risk
- Visualizes results using graphs
- Provides recommendations for best focus area

# Technologies Used
- Python
- Pandas
- Matplotlib

# Example Output
The system ranks applications and recommends the most effective aerospace AI use case based on weighted scoring

# Real-World Relevance
This tool can support aerospace researchers and engineers in prioritizing AI projects for aviation and space missions.
